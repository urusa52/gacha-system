# 캐릭터 일러스트 폴더

여기에 이미지를 넣고, `src/config/characters.data.js`의 해당 캐릭터
`image` 필드에 경로를 적으면 복원 화면에 표시됩니다.

예:
  image: "assets/characters/wuxia_swordsman_01.png"

- 권장 비율: 세로형 (3:4 근처). object-fit: cover로 잘려 표시됩니다.
- 이미지가 없으면(image: null) CSS 실루엣이 자동 표시됩니다.
- AI 생성 이미지 사용 시, 제출용 "AI 활용 기술 문서"에 생성 도구와
  프롬프트를 기록해 두세요.
